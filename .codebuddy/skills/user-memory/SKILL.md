## 工作流偏好

- ✅ 大方案 → 先出多方案选项，用户确认后再落地
- ✅ 涉及 UI → 先确认设计方向再编码
- ✅ 后端服务 → 职责单一，不跨服务耦合
- ❌ 不主动 git commit
- ❌ 不做拍脑袋的设计，给出理由
- ⏳ 用户审查时间：晚间 22:00-00:00 活跃
