## 问题清单

### 1. 没有头像，需要一个默认头像
- **问题描述：** 用户登录后没有显示默认头像
- **影响范围：** 所有页面右上角用户头像区域
- **解决方案：** 
  - 添加一个卡通风格的默认头像图片
  - 在用户数据没有 avatar 时使用默认头像
- **优先级：** P1

### 2. 点击退出没调用退出接口
- **问题描述：** 点击退出登录按钮后，没有调用后端 logout 接口
- **影响范围：** 用户退出功能
- **解决方案：**
  - 检查退出按钮的事件处理
  - 添加 logout API 调用
  - 清除本地 token 并跳转到登录页
- **优先级：** P0（最高）

### 3. 用户列表没显示
- **问题描述：** 用户管理页面列表为空或不显示
- **影响范围：** 用户管理模块
- **解决方案：**
  - 检查用户列表 API 调用
  - 检查数据绑定和渲染逻辑
  - 修复可能的接口路径或参数问题
- **优先级：** P0（最高）

### 4. 菜单没切换
- **问题描述：** 点击菜单项后，菜单没有高亮切换或路由没有变化
- **影响范围：** 全局导航菜单
- **解决方案：**
  - 检查菜单组件的路由绑定
  - 检查 activeKey 或 selectedKeys 逻辑
  - 修复路由配置
- **优先级：** P1
