# 变变 — 交互设计规范 v1.0

> 版本：v1.0  
> 目标设备：手机（iPhone 优先，适配主流 Android）  
> 设计基准：375×812（iPhone X 逻辑分辨率）

---

## 一、全局交互规则

### 1.1 导航模型

```
                     ┌──────────┐
                     │   首页    │
                     └────┬─────┘
                          │
            ┌─────────────┼─────────────┐
            ▼             │             ▼
      ┌──────────┐        │      ┌──────────┐
      │  创作页   │        │      │ 我的变变  │
      └────┬─────┘        │      └──────────┘
           │              │
           ▼              │
      ┌──────────┐        │
      │ AI变身页  │        │
      └────┬─────┘        │
           │              │
           ▼              │
      ┌──────────┐        │
      │  结果页   │────────┘
      └──────────┘
```

- 首页 ↔ 创作页：右滑进/左滑出
- 首页 ↔ 我的变变：右滑进/左滑出
- 创作页 → 变身页：底部弹出（Modal）
- 变身页 → 结果页：变身成功后自动替换
- 结果页 → 首页：「再创一个」→ Pop 回首页

### 1.2 手势系统

| 手势 | 场景 | 行为 |
|------|------|------|
| 右滑 | 创作页、我的变变 | 返回上一页 |
| 点击素材 | 素材栏 | 添加到画布中心 |
| 长按素材 | 素材栏 | 开始拖拽（iOS 拖拽 API） |
| 拖拽 | 画布上 | 移动素材位置 |
| 双指缩放 | 画布上、结果页 | 缩放素材/角色 |
| 双指旋转 | 画布上 | 旋转素材 |
| 双击素材 | 画布上 | 删除素材（弹出确认） |
| 下拉 | 首页、我的变变 | 无刷新（MVP 不实现） |

### 1.3 触控热区

- 最小可点击区域：44×44pt（符合 Apple HIG）
- 按钮最小高度：48pt
- 按钮间距：≥ 8pt

### 1.4 键盘行为

- 首页：无键盘
- 创作页：点击「描述输入框」→ 键盘弹起 → 画布上移避免遮挡
- 结果页：无键盘
- 键盘收起时点击画布区域

```
┌────────────────────────────────────────────┐  812pt
│  Safe Area Top                             │
│                                            │

│  │                                     │   │
│  │         画布区域                      │   │
│  │    （触摸操作，自由拖拽）               │   │

│  │                                     │   │
│  └─────────────────────────────────────┘   │
│                                            │
│  ┌─────────────────────────────────────┐   │  48pt
│  │ 💬 描述一下你的作品？（选填）           │   │
│  └─────────────────────────────────────┘   │
│                                            │
│  ┌─────────────────────────────────────┐   │  56pt
│  │         🎲 随机素材      变变！ ✨    │   │
│  └─────────────────────────────────────┘   │
│                                            │
└────────────────────────────────────────────┘
```

**交互细节：**

| 组件 | 行为 |
|------|------|
| 顶部栏 | 「← 返回」点击 → 自动保存草稿 → 返回首页；「变变！」跳转按钮 |
| 素材 Tab | 横向滚动 Tab；点击切换分类；选中态：橙色下划线 + 加粗 |
| 素材列表 | Tab 下方出现横向滚动素材列表（二行）；点选/拖拽到画布 |
| 画布 | 空白状态显示虚线边框 + 引导文案「拖拽素材或画一画～」；有内容后虚线消失 |
| 画布上的素材 | 选中态：四角 resize 手柄 + 旋转手柄 + 删除按钮 |
| 描述输入 | 点击弹键盘；占位文「比如：我画了一只会飞的小鱼」；字数限制 50 字 |
| 底部栏 | 「🎲 随机素材」：随机添加一个素材到画布；「变变！✨」：触发 AI 变身 |
| 变变按钮 | 画布为空时置灰不可点击；画布有内容时主色高亮 + 呼吸光晕 |

**画布操作详情：**

| 操作 | 手势 | 反馈 |
|------|------|------|
| 选中素材 | 单击 | 出现蓝色选中框 + 四角手柄 |
| 移动素材 | 拖拽 | 跟随手指 + 轻微吸附效果 |
| 缩放素材 | 双指捏合 | 实时跟随；限制 0.3x-3x |
| 旋转素材 | 双指旋转 | 实时跟随；15° 吸附 |
| 删除素材 | 双击素材 或 点选中态 × 按钮 | 弹出 toast「已删除」；撤销 3 秒 |
| 复制素材 | 长按素材 → 弹出菜单「复制」 | 复制体偏移 20pt 出现在右上 |
| 涂鸦 | 手指在画布上画（无素材区域） | 默认笔刷：圆头 4pt，颜色跟随上次选色 |
| 撤销涂鸦 | 双指轻点 | 撤销最后一步涂鸦笔画 |

**草稿保存逻辑：**
- 任何修改后 2 秒无操作 → 自动保存
- 保存内容：素材位置/大小/旋转 + 涂鸦数据 + 描述文字
- 退出确认：若 5 秒内有过修改且未保存 → 弹窗「正在保存中…」

```
│   ┌────────────────────────────────────┐   │

│   │  皮克斯的动画师平均需要 1 周制作     │   │
│   │  一个角色，但变变只需要几秒钟！       │   │
│   └────────────────────────────────────┘   │
│                                            │
│                                            │
└────────────────────────────────────────────┘
```

**交互细节：**

| 时间节点 | 状态 | 表现 |
|---------|------|------|
| 0-3 秒 | 正常 | 显示「变身中...」+ 进度条 0-40% |
| 3-10 秒 | 正常 | 进度条 40-90%，速度非线性（先快后慢） |
| 10+ 秒 | 等待 | 显示「还在努力中，画得越用心变得越好看哦」 |
| 15+ 秒 | 等待 | 播放趣味小知识轮播（共 5 条，每 3 秒切换） |
| 30 秒超时 | 失败 | 显示失败原因 + 「再变一次」按钮 |

**动效设计：**
- 原画缩略图 → 脉冲式缩放呼吸（放大 1.05x → 回 1.0x，周期 1.5s）
- 进度条：主色橙色，渐变背景，圆角端点
- 变身成功瞬间：全屏白色闪光（100ms）→ 结果页弹簧弹出

## 三、动效规范

### 3.1 动效参数表

| 动效名称 | 曲线 | 时长 | 应用场景 |
|---------|------|------|---------|
| ease-out | cubic-bezier(0,0,0.58,1) | 200ms | 按钮回弹、页面滑入 |
| spring | spring(1, 100, 10) | ~400ms | 角色弹出、变身成功 |
| pulse | ease-in-out 循环 | 1500ms/周期 | 原画呼吸、主按钮光晕 |
| slide-in-right | ease-out | 300ms | 页面切换 |
| fade-in | ease-out | 200ms | 元素渐现 |

### 3.2 关键场景动效分解

**变身全流程动效时间线：**

```
t=0ms     点击「变变！」
t=0-200ms 按钮缩小 0.95x → 弹簧回弹到 1.0x
t=200ms   创作页右滑退出 + 变身页滑入（同期）
t=500ms   变身页动画开始：原画缩略图脉冲呼吸
t=1000ms  进度条开始填充（非线性）
t=3-10s   进度条 40-90%（先快后慢）
t=成功    全屏白色闪光 100ms
t=+100ms  结果页弹簧弹出（scale 0.8→1.05→1.0）
t=+600ms  角色卡片弹出完成
t=+800ms  底部按钮依次淡入（间隔 100ms）
```

## 四、异常与边界状态

### 4.1 网络异常

| 场景 | 表现 |
|------|------|
| 变身中网络断开 | 进度条暂停 → 10秒后弹 Toast「网络走丢了，检查一下再试试」+「再变一次」按钮 |
| App 被杀 | 草稿自动保存（沙盒文件）；重启后首页显示「上次创作」恢复条 |

### 4.2 存储异常

| 场景 | 表现 |
|------|------|
| 存储空间不足 | Toast「手机空间不够了，清理后再来变吧」 |
| 历史满 20 条 | 自动删旧 + Toast 提示 |

### 4.3 权限异常

| 场景 | 表现 |
|------|------|
| 相册权限未授权 | 点「保存到相册」→ 弹系统授权 → 拒绝后引导到「设置」 |
| 相册权限被拒绝 | 第二次点保存 → 「保存失败：请在设置中允许访问相册」+ 「去设置」按钮 |

### 4.4 变身失败

| 原因 | 用户提示 | 后续动作 |
|------|---------|---------|
| 画布空白 | 「画布还是空的哦～放点素材上去吧」 | 返回创作页 |
| 图片不清晰 | 「这张画不太清楚～可以再画一张试试吗？」 | 「再画一张」按钮 |
| 内容不适宜 | 「换一张画试试吧，变变最喜欢可爱的画了！」 | 「再画一张」按钮 |
| API 超时 | 「变身超时了～再试一次？」 | 「再变一次」按钮 |
| API 报错 | 「出了点小问题，再试一次？」 | 「再变一次」按钮 |

## 五、无障碍要求（MVP 最低标准）

| 要求 | 说明 |
|------|------|
| 文字对比度 | 所有文字与背景对比度 ≥ 4.5:1（WCAG AA） |
| 触控目标 | 所有可交互元素 ≥ 44×44pt |
| 字体大小 | 不支持系统字体缩放时可接受（MVP） |
| VoiceOver | MVP 不做（后期补） |
