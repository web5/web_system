## 一、产品定位

**一句话**：不是教孩子画画，是用 AI 把孩子的涂鸦/拼贴变成可爱的 3D 角色——给孩子的创造力一个「被看见」的正反馈。

| 维度 | 说明 |
|------|------|
| 核心用户 | 6-10 岁孩子的家长（决策者），孩子是实际使用者 |
| 使用场景 | 孩子无聊时打开 App → 用素材拼贴/涂鸦捣鼓 5-10 分钟 → 一键「变变」→ 得到专属角色 → 下次还想玩 |
| 核心价值 | 让孩子的想象力**可视化**、**可触摸**、**可炫耀** |
| 品类定义 | AI 儿童创意工具（非教育工具，是创造力放大器） |

### 用户故事

- **作为孩子**，我希望拼一拼贴一贴，就能变出我自己的角色，**以便**我觉得自己很厉害、想给妈妈看。
- **作为家长**，我希望孩子用手机不只是刷视频，**以便**他能主动创作、我能看到他的想象力在成长。
- **作为家长**，我希望孩子完成的作品能保存/分享，**以便**我记录他的成长，也能在朋友圈获得点赞。

## 三、MVP 功能范围

### ✅ 必须有（MVP）

| 功能 | 说明 |
|------|------|
| 首页 | 「开始创作」+「我的变变」两个主入口 |
| 创作页 | 基础素材库（贴纸/形状/背景）+ 简单涂鸦画笔 |
| 画布操作 | 拖拽素材、调整大小/旋转、删除、简单涂鸦 |
| AI 变身 | 上传画布 → 调用 AI 接口 → 返回 3D IP 角色 |
| 结果页 | 原画对比 + AI 角色展示 + 保存到相册 |
| 变变历史 | 本地缓存最近 20 张（纯本地，不登录） |
| 草稿恢复 | 退出后再次进入，恢复上次未完成的画布 |

### ❌ 不做（MVP）

- ❌ 登录/注册（纯本地使用）
- ❌ 付费/订阅
- ❌ 画板过于复杂（不要多层、不要高级笔刷）
- ❌ 一键分享到朋友圈（太重，改「保存到相册」）
- ❌ IP 交易/社区
- ❌ 多风格选择（MVP 固定皮克斯风格）
- ❌ 云端存储

**三种状态：**

| 状态 | 触发条件 | 表现 |
|------|---------|------|
| ✅ 成功 | AI 正常返回 | 自动跳转结果页 |
| ⏳ 等待中 | 超过 10 秒 | 显示「还在努力中，画得越用心变得越好看哦」 |
| ❌ 失败 | 接口超时/报错 | 显示失败原因 + 「再变一次」按钮 |

**失败原因与提示：**
- 图片不清晰 → 「这张画不太清楚～可以再画一张试试吗？」
- 内容不适宜 → 「换一张画试试吧，变变最喜欢可爱的画了！」
- 网络异常 → 「网络走丢了～检查一下再试试？」

**交互规则：**
- 点击「重新变」：用同一张原画再次调用 AI，生成略有差异的结果
- 限制：「重新变」最多 3 次（含首次），防止刷 API
- 达到上限后，「重新变」按钮置灰，显示「今天变够啦，明天继续！」

## 五、AI Prompt 工程

详见独立文档：[AI-Prompt-变变.md](./AI-Prompt-变变.md)

核心原则：
- 固定风格：皮克斯 3D 卡通
- 固定输出：1024x1024 正方形、白底、全身正面
- 控制参数：保留原画配色 70%、提升质感 30%
- 负面约束：不添加原画中没有的显著元素

## 七、风险与应对

| 风险类型 | 具体风险 | 概率 | 应对方案 |
|---------|---------|------|---------|
| AI 内容安全 | 孩子画作含不适宜内容 | 中 | API 前加内容审核（阿里绿网/腾讯天御） |
| AI 效果差 | 画太抽象，AI 理解不了 | 高 | 「重新变」按钮 + 最多 3 次重试 + 画前引导 |
| API 延迟 | 响应 5-30 秒不稳定 | 高 | 前端进度条 + 预估时间 + 超过 15 秒给趣味提示 |
| 本地存储溢出 | 20 张历史存满 | 中 | 自动清理最早作品 + 引导保存到相册 |
| 竞品追赶 | 即梦/通义出儿童模式 | 中 | 差异化：我们做「孩子创造的角色」，不做「图生图工具」 |
| 版权争议 | AI 生成的角色版权归属 | 中 | MVP 阶段生成结果免费归用户；长期需法律确认 |

## 八、设计系统（Design System）

### 色彩系统

| 角色 | 色值 | 用途 |
|------|------|------|
| 主色-魔法橙 | `#FF8C42` | 主按钮、品牌识别 |
| 辅色-天空蓝 | `#4ECDC4` | 辅助元素、链接 |
| 背景-暖白 | `#FFF8F0` | 全局背景，护眼不刺眼 |
| 成功-薄荷绿 | `#7ED957` | 成功状态 |
| 文字-深灰 | `#333333` | 主要文字 |
| 文字-中灰 | `#888888` | 辅助文字 |

### 圆角系统

| 元素 | 圆角 |
|------|------|
| 大卡片 | 24px |
| 按钮 | 16px |
| 小标签 | 12px |

### 字体系统

| 层级 | 字号 | 字重 | 用途 |
|------|------|------|------|
| H1 | 28pt | Bold | 页面主标题 |
| H2 | 22pt | Bold | 区块标题 |
| Body | 16pt | Regular | 正文、按钮文字 |
| Caption | 13pt | Regular | 提示文字 |

### 动效原则

| 场景 | 动效 | 时长 |
|------|------|------|
| 按钮按下 | 缩放至 0.95x | 100ms |
| 按钮弹起 | 弹簧回弹 | 300ms |
| 变身进度 | 波浪形呼吸动画 | 持续 |
| 变身成功 | 弹出 + 缩放 + 粒子特效 | 500ms |
| 页面切换 | 从右往左滑入 | 300ms |

## 九、技术提示（给开发）

### AI 接口选择

| 选项 | 优势 | 劣势 | 推荐 |
|------|------|------|------|
| 通义万相 图生图 | 阿里生态，稳定 | 儿童画支持需验证 | ✅ 首选 |
| 即梦 图生图 | 字节生态，效果好 | 接入门槛高 | 备选 |
| Stable Diffusion API | 可定制化高 | 运维成本高 | 不推荐 MVP |

### Prompt 模板

见 [AI-Prompt-变变.md](./AI-Prompt-变变.md)

### 本地存储方案

| 存储内容 | 方案 | 上限 |
|---------|------|------|
| 变变历史（缩略图） | UserDefaults / SharedPreferences | 20 条 |
| 草稿画布 | UserDefaults（序列化） | 1 条（覆盖写） |
| 原图/结果图 | App 沙盒 Document 目录 | 系统限制，不主动管理 |
