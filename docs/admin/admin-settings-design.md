# 后台管理系统 - 系统设置产品方案

> 版本：v1.0 | 产品：科豆 AI 管理后台 | 日期：2026-06-13

---

## 一期：上线必备（基本设置 + 安全策略 + 功能开关）

### 1. 基本设置

| 字段 | 类型 | 说明 | 默认值 |
|------|------|------|--------|
| 站点名称 | 文本 | 显示在浏览器标题、登录页 | 科豆 AI |
| 站点描述 | 文本域 | SEO description | 科豆 AI 少儿教育平台 |
| 联系邮箱 | 邮箱 | 用户反馈/通知发件人 | admin@kedouai.com |

**存储**：`system_config` 表，key-value 结构。

### 2. 功能开关

| 开关项 | 说明 | 默认值 | 影响范围 |
|--------|------|--------|----------|
| 微信扫码登录 | 控制 Portal 扫码登录是否可用 | 开 | portal 登录页 |
| 账号密码登录 | 控制账号登录是否可用 | 开 | portal + admin 登录页 |
| 用户自助注册 | 允许用户自行注册账号 | 开 | portal 注册表单 |
| 小程序端入口 | 小程序是否对外开放 | 开 | mini-app 首页 |
| 评论功能 | 课程评论是否开放 | 关 | portal 课程区 |

**存储**：同 `system_config` 表，value 为 `"1"` / `"0"`。

### 3. 安全策略

| 字段 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| 密码最小长度 | 数字 | 8 | 注册/重置密码时校验 |
| 密码必须含数字 | 开关 | 开 | - |
| 密码必须含大写字母 | 开关 | 关 | - |
| 密码必须含特殊字符 | 开关 | 关 | - |
| 登录失败锁定阈值 | 数字 | 5 | 连续失败 N 次后锁定 |
| 锁定时间（分钟） | 数字 | 30 | 锁定时长 |
| Token 有效期（天） | 数字 | 7 | access_token 过期时间 |

**存储**：同 `system_config` 表。

## 二期：运营支撑（通知配置 + 存储配置 + 操作日志）

### 4. 通知配置

#### 邮件通知（SMTP）

| 字段 | 说明 | 示例 |
|------|------|------|
| SMTP 服务器 | 发件服务器地址 | smtp.qq.com |
| SMTP 端口 | 端口号 | 465 |
| 发件人邮箱 | 通知邮件 From 地址 | admin@kedouai.com |
| 授权码 | SMTP 密码/授权码 | ●●●●●●●● |
| 加密方式 | SSL / TLS / 无 | SSL |

#### 微信模板消息（后期）

| 字段 | 说明 |
|------|------|
| AppID | 微信公众号/小程序 AppID |
| 模板ID | 微信模板消息 ID |

**存储**：`system_config` 表（敏感字段加密存储）。

### 5. 存储配置

| 字段 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| 存储方式 | 单选 | 本地 | 本地 / 阿里云OSS / 腾讯COS |
| Bucket | 文本 | - | 云存储桶名称 |
| Region | 文本 | - | 地域 |
| AccessKey | 文本 | - | 访问密钥（加密存储） |
| 上传大小限制(MB) | 数字 | 10 | 单文件最大上传大小 |

### 6. 操作日志

| 字段 | 说明 |
|------|------|
| 操作人 | 管理员用户名 |
| 操作类型 | 登录 / 退出 / 创建用户 / 修改设置 / 删除 |
| 操作目标 | 被操作的对象（用户名/配置项） |
| 操作时间 | 精确到秒 |
| IP 地址 | 操作时 IP |

**功能**：
- 按时间范围筛选
- 按操作人筛选
- 按操作类型筛选
- 分页展示（每页 20 条）
- 日志自动保留 90 天

**存储**：`operation_logs` 独立表。

## 三期（规划）

| 模块 | 内容 |
|------|------|
| 用户协议 | 富文本编辑用户协议 & 隐私政策 |
| 数据维护 | 缓存清理 / 数据库备份 / 版本升级 |
| 角色权限 | 管理员角色管理 & 权限矩阵 |
| 系统监控 | CPU / 内存 / 磁盘 / API 调用量 |

---

## API 设计

### `GET /api/admin/settings`
获取所有配置项。

### `PUT /api/admin/settings`
批量更新配置项。
```json
{
  "site_name": "科豆 AI",
  "wechat_login_enabled": "1",
  "pwd_min_length": "8"
}
```

### `GET /api/admin/logs?page=1&pageSize=20&operator=&type=&start=&end=`
查询操作日志。

---

## 前端路由

```
/admin/settings
  ├── /basic       → 基本设置
  ├── /features    → 功能开关
  ├── /security    → 安全策略
  ├── /notify      → 通知配置
  ├── /storage     → 存储配置
  └── /logs        → 操作日志
```
